MessageHandler.h File Reference

Header file used to declare the MessageHandler class. The MessageHandler class is designed using a Singleton design pattern so that one object can be used throughout the code for communication with the embedded system or simulated embedded system, regardless of our location within the code. The communication is handled with separate threads for sending messages to the embedded system, receiving messages from the embedded system, and simulating the embedded system. Using the object, a user can simply call a function to send a message and the receive the appropriate data as a result. More...

Detailed Description

Header file used to declare the MessageHandler class. The MessageHandler class is designed using a Singleton design pattern so that one object can be used throughout the code for communication with the embedded system or simulated embedded system, regardless of our location within the code. The communication is handled with separate threads for sending messages to the embedded system, receiving messages from the embedded system, and simulating the embedded system. Using the object, a user can simply call a function to send a message and the receive the appropriate data as a result.
