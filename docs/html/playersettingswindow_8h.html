Detailed Description

Header file used to declare the playersettingswindow class.

Author
Natalia Bohulevych The playersettingswindow class is responsible for displaying the GUI that pertains to the player class. It allows the user to select different settings for each of the data members in the player class. This class works on the player objects that are passed to it, in the context of this project, this would be the tplayer objects held by the main menu window. Should the user hit accept on the GUI interface, the chosen settings will be applied to the passed player objects.
